# hhParser

Проект, предназначенный для парсинга сайта с вакансиями hh.ru, а также сбором некоторых статистик.

## Запуск
Все команды необходимо выполнять, находясь в корневой директории проекта

Для запуска виртуального окружения:
```
./Scripts/activate
```
(на линуксе это возможно не так)

Для скачивания всех зависимостей:
```
pip install .
```

Для запуска программы:
```
./Scripts/python.exe ./src/main.py
```

## Файл конфигурации
Далее описание переменных в файле конфигурации ```config.json```

```search```: Поисковый запрос для парсинга вакансий (напр. ML engineer) (Кириллица не поддерживается)

```filter_experience```: Фильтр для опыта работы, выберите один из следующих вариантов (число от 1 до 5):
- 1) Не имеет значения
- 2) Без опыта
- 3) От 1 до 3 лет
- 4) От 3 до 6 лет
- 5) Более 6 лет

```num_vacancies```: Количество вакансий для парсинга.

```concurrent_requests```: Максимальное количество асинхронно обрабатывающихся http запросов.

```make_graphs```: Создавать графики или нет (1 или 0)
